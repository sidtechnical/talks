DIY Tools suggestion

Mobile Phone Metadata

  • Bandicoot:An open-source Python toolbox to analyze mobile phone metadata.
  • Easy-to-use to analyze mobile phone metadata.
  • Mobile app to capture the call data records.
  • Securely encrypt, anonymize and export the captured data from phone to computer for further analysis.
  • Built-in visualization tools to derive social graph, mobility pattern, and interactions.
  • To understand the behavioural indicators.

Bandicoot

Reference : Bandicoot

Geolocation Metadata

  • Geocreepy:An open-source Geolocation OSINT Tool.
  • Geolocation information gathering through social networking platforms.
  • A desktop app wich allows to put together geolocation traces from social media.
  • Works with Facebook, Twitter, Instagram, Flickr, ……
  • search filtering based on exact location and/or date.
  • To understand yours or anyone's geo-persona using built-in APIs.

Geocreepy

Reference : GeoCreepy

Tweet Metadata

  • tweets_analyzer:An open-source tool for scraping Tweets metadata and analysis.
  • An esy-to-use command line tool.
  • Distinguishes “corporate” accounts (only used during working hours) and guesses how many users are interacting with the account.
  • Geolocations, most used hashtags, languages used.
  • Timezone and source (device).
  • Simple but elegant.

x0rz's tweets_analyzer

Reference : tweets_analyzer

Mobile Sensor Metadata

  • Funf Journal:An open-source android app to unserstand and give contextual meaning to mobile sensors.
  • Bluetooth, Wi-fi, Cellular connection, Location, Battery, Accelerometer, Internal temperature.
  • Screen OFF/On and general phone usage patterns.
Reference : FUNF Journal

Best use of Metadata

  • InformaCam - CameraV:An open-source tool for journalists.
  • An Android app that enables users to inflate image and video with extra metadata.
  • GPS coordinates, altitude, compass bearing, light meter readings, the signatures of neighboring devices, cell towers, and wifi networks.
  • Creates a digital snapshot of the environment in which the media was captured.
  • Encrypting and signing the metadata assures that it is verifiable and tamper-proof.
